# 계정 삭제 요청 안내

**서비스**: 영영: 영어회화 영작연습  
**개발사**: 트리스트

## 요청 절차

1. 삭제를 원하는 계정의 **이메일**로 앱에 로그인합니다.
2. **support@treestcorp.com** 으로 아래와 같은 이메일을 보내주세요.
    - 제목: **계정 삭제 요청**
    - 본문: 삭제 요청 사유나 추가 설명 (선택 사항)
3. 요청 확인 후 계정 및 관련 데이터는 아래 정책에 따라 삭제됩니다.

## 데이터 삭제 정책

| 데이터 유형       | 처리 방식                          | 보관 기간                  |
|-------------------|------------------------------------|----------------------------|
| 인증 정보         | 삭제 요청 확인 후 즉시 삭제        | -                          |
| 학습 진행 정보    | 삭제 요청 확인 후 즉시 삭제        | -                          |
| 구매 내역         | 관련 법령에 따라 보관 후 삭제      | 법령에 따라 최대 5년       |

- 구매 내역의 경우 세금·회계 관련 법규(전자상거래 등에서의 소비자보호에 관한 법률, 국세기본법 등)에 따라 일정 기간 보관 후 파기됩니다.
- 삭제 요청 처리 완료 시 별도 이메일로 안내드립니다.

문의사항이 있으시면 언제든 **support@treestcorp.com** 으로 연락 주세요.
